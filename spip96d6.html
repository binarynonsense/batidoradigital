Crea páginas web sin gastar un céntimo

Diseño web con herramientas gratuitas

Lunes 26 de Septiembre de 2005, por Álvaro García

Diseñar páginas web atractivas ya no sólo está al alcance de profesionales con acceso a costosos programas como Dreamweaver o Photoshop, o con el dinero suficiente para pagar espacio en un servidor. La enorme cantidad de tutoriales que hay en internet, el movimiento de software libre, los programas freeware, los servidores gratuitos y los dominios gratuitos, permiten aprender cómo crear una web, diseñarla, programarla y hospedarla en un servidor sin gastar un céntimo.

En este artículo voy a tratar de recopilar una lista de herramientas gratuitas, software libre (aún mejor) y sitios webs con los que poder llevar a cabo todo el proceso de creación de páginas web sin invertir ni un euro.

A pesar de ser gratuitos muchos de los programas aquí mostrados son de calidad igual o superior a sus alternativas comerciales. El hecho de no costar dinero no implica que sean de mala calidad, en especial aquellos que son software libre, que tienen detrás una gran comunidad de programadores.

Páginas sobre diseño web:

Antes de nada debemos saber cómo va a ser nuestra web, su diseño, qué lenguajes vamos a usar, su aspecto visual. Una vez hecho esto podremos ponernos manos a la obra. Para empezar a conocer qué recursos tenemos a nuestra disposición, encontrar ideas, consejos y tutoriales aquí van unas direcciones de páginas sobre diseño web:

http://www.desarrolloweb.com/

http://www.maestrosdelweb.com/index.php

http://www.webtaller.com/

http://www.webestilo.com/

http://www.webexperto.com/

http://www.htmlweb.net/

http://programacion.com/

A continuación dividiré los recursos en varios apartados relacionados con áreas concretas del diseño web:

Diseño Gráfico

 The Gimp - un programa de retoque fotográfico y creación de imágenes al estilo Photoshop. De lo mejor.

 Inkscape - Si quieres utilizar imágenes vectoriales este es un gran programa.

 EasyRGB - Página web que nos calcula qué colores armonizan con uno dado. No es perfecto pero puede ser útil a los principiantes que duden qué colores usar en su página.

 MapThis - Programa para crear mapas de imágenes.

Editores

 Arachnophilia - Editor de texto especializado en HTML, aunque va por la versión 5 recomiendo la 4, más sencilla y que no necesita java para funcionar

 Crimson Editor - Editor de texto para programadores, resalta la sintaxis de HTML, PHP, C/C++, Perl, Java, Matlab y LaTeX

 Nvu - Editor visual de páginas web tipo Dreamweaver pero software libre. Muy útil.

HTML

HTML (HyperText Markup Language - lenguaje de marcación de hipertexto) es un lenguaje de marcas usado para la creación de páginas webs.

 webmaestro- Este manual está pensado para guiar tanto a quien desee construir su propia página del Web, partiendo desde los conceptos iniciales, como a quien quiera ampliar sus conocimientos

 HTML a fondo - Manual de HTML de Desarrolloweb

 Manual de HTML - Manual de HTML de WebEstilo

PHP

PHP es un lenguaje de programación interpretado y libre muy utilizado en la creación de páginas webs como complemento de HTML con multitud de posibilidades: páginas web dinámicas, comunicación con bases de datos, lectura y escritura de ficheros, gestión de usuarios, cookies...

 php.net - Página oficial de PHP, si necesitas consultar para qué sirve un comando en php este es el mejor sitio.

 PHP a fondo - Manual de PHP de Desarrolloweb

 Manual de PHP - Manual de PHP de WebEstilo

 PHP Freaks - Web dedicada a PHP con tutoriales, artículos, código...

CSS

CSS (Cascading Style Sheets - hojas de estilo en cascada) es un lenguaje formal usado para definir la presentación de un documento HTML.

 CSS a fondo - Manual de CSS de Desarrolloweb

 Manual de CSS - Manual de CSS de WebEstilo

 css Zen Garden - demostración de lo que se puede hacer con CSS

 Maquetar una página con CSS - artículo muy interesante

 CSS Layout Techniques

 CSS Vault

Javascript

JavaScript es un lenguaje interpretado orientado a las páginas web para realizar tareas y operaciones en el marco de la aplicación cliente.

 Javascript a fondo - Manual de javascript de Desarrolloweb

 Manual de Javascript - Manual de javascript de WebsEstilo

recopilaciones de scripts en javascript:

 Javascript Kit

 Hotscripts

 The Javascript Source

Gestores de contenido

Los gestores de contenido o CMS (Content Management System) permiten la creación y administración de contenidos principalmente en páginas web. Suelen consistir en una interfaz que controla una o varias bases de datos donde se aloja el contenido del sitio. Existen multitud de CMS, y muchos de ellos son software libre, veamos algunos de los más destacados:

 WordPress - de lo mejor para crear un blog

 SPIP - con el que está gestionada la Batidora Digital

 CMSimple - un CMS que no necesita base de datos

 XOOPS - creación de portales

 DocuWiki - crea tu propia wiki

 PhpBB - para crear un foro

Si quieres comprobar cómo funciona alguno de ellos antes de instalarlo, o conocer más, visita:

 open source cms

Para ver una lista con más CMS open source ve a:

 Open Source Content Management System List

Plantillas prefabricadas

Si lo tuyo no es el diseño gráfico siempre puedes recurrir a las plantillas web prefabricadas. Diseños ya hechos que la gente pone a disposición de los demás.

 Open Souce Web Design - Recopilación de plantillas open source que puedes usar libremente

Hospedaje web gratuito

 lycos - 50 megas de espacio, PHP 4, MySQL...

 host.sk - 50 megas de espacio, PHP 5, MySQL...

Clientes FTP

Para subir los archivos de la página web al servidor puedes hacer uso del webFTP que algunos ofrecen o, mejor aún, de un cliente FTP:

 Filezilla

Dominio gratuito

 Dominio tk - dominio de la isla de Tokelau que ofrece una versión gratuita.

Estándares y recomendaciones

 World Wide Web Consortium (W3C) - El World Wide Web Consortium (W3C) desarrolla especificaciones, recomendaciones, software y herramientas para la web. Si quieres que tu web funcione bien en la mayoría de navegadores es conveniente seguir las recomendaciones y estándares del W3C.

Navegadores

Para asegurarse de que todos los visitantes vean bien la página web hay que probarla antes con el mayor número de navegadores posibles.

 Mozilla Firefox - probablemente el mejor navegador que existe actualmente, y que además es software libre y se adapta a los estándares del W3C

 Opera

Servidor de pruebas

Si usas PHP o Perl en tus páginas puede que te interese instalar un servidor web en tu PC para ir probando lo que haces sin tener que subirlo cada dos por tres a tu servidor en internet para ver si funciona como quieres. De nuevo podemos hacer uso del software libre:

 XAMPP - Instalar un servidor web en nuestro ordenador y configurar PHP, Mysql y/o Perl puede ser complicado, pero XAMPP está aquí para facilitarnos la tarea. Contiene un servidor Apache, PHP, Mysql y Perl entre otras cosas. Simplemente hay que descargarlo, descomprimirlo y ya está.
