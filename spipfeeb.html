Knoppix: Linux fácil al alcance de todos

pruébalo desde el cd o instálalo fácilmente en tu disco duro

Lunes 4 de Abril de 2005, por Álvaro García

Tener que instalar linux para probarlo siempre ha sido una traba para los que querían iniciarse en este interesante mundo. Pero esto se acabó desde que se crearon las distribuciones live-cd. Las distribuciones live-cd son sistemas operativos linux que no necesitan instalarse en el disco duro para funcionar. Se arranca el ordenador con el cd de la distribución en tu lector de cd/dvd y...ya está, puedes disfrutar de un sistema linux completo.

Knoppix es una de las distribuciones pioneras en el mundo de las live-cd, y para muchos la mejor de ellas.

Esta distribución de linux trae un montón de programas libres para que puedas empezar a usar linux y no eches casi nada de menos. Entre los más destacados están: OpenOffice, una suite ofimática a la altura de las mejores, con procesador de textos, hoja de cálculos, presentaciones... The Gimp, un programa de retoque fotográfico de igual calidad que muchos programas comerciales; el navegador Mozilla; un montón de programas para programadores (linux es uno de los mejores entornos de programación, ya que casi todos los lenguajes tienen entornos de programación libres dedicados a ellos; un entorno de ventanas, KDE, para que los acostumbrados a windows se desenvuelvan perfectamente por el sistema sin necesidad de utilizar la temida línea de comandos, aunque recomiendo aprender cuanto antes a utilizar los comandos en linux, porque se pueden hacer multitud de cosas con ellos :);...y muchos más programas para casi todas las necesidades del usuario, incluidos algunos juegos.

Lo primero que necesitas para disfrutar de esta maravilla es el cd de Knoppix: puedes obtenerlo de su web: www.knoppix.org o en alguna revista de linux, de vez en cuando traen la última versión de knoppix.

Pruébalo desde el cd, arranca el ordenador con el cd en la unidad lectora, espera a que arranque linux, prueba los programas, navega, explora y si te gusta te recomiendo instalarlo. Es más cómodo tener el sistema en tu disco duro, te olvidas del cd, puedes instalar programas nuevos (el mundo del software libre es inmenso, y siempre hay nuevos programas que probar), puedes desinstalar los que no uses,...

Instalar Knoppix en tu disco duro

Si has decidido instalar Knoppix en tu disco duro, no te preocupes, es bastante fácil. Y si tienes dudas, internet está lleno de artículos sobre cómo hacerlo, y foros sdonde preguntar a gente más experta (la comunidad del software libre está formada por gente muy maja :) que responde a las preguntas de lo novatos rápidamente e intenta ayudarse entre si)

yo voy a describir brevemente los pasos que seguí yo cuando la probé en mi disco duro. Es una magnífica distribución que recomiendo a todos los novatos. Tiene todo lo necesario para empezar a aprender linux. Una vez dominado, quizá quieras probar otras distribuciones más completas. Yo recomiendo por potencia y sencillez Suse professional, aunque aquí cada uno tiene su favorita.

Pasos para instalar Knoppix:

Una vez arrancado el ordenador con el cd de knoppix durante el arranque nos sale una línea que pone

boot:

aquí podemos indicarle en que idioma queremos que arranque el sistema: boot:knoppix lang=es

tras esto se carga el gestor de ventanas KDE.

Abrimos una shell, una ventana de comandos, y escribimos:

knx2hd

(si no funciona otras opciones son

knoppix-install

knx-hdinstall

)

con esto se arranca el proceso de instalación siguiendo una serie de pasos simples.

Para instalar knoppix necesitamos al menos 2 particiones, una para el sistema en sí y otra de swap. para la primera recomiendo una partición de 3 gigas, para la segunda, el doble de tu memoria RAM, aunq si tienes más de 384MB valdrá con un partición de igual tamaño que la RAM.

Pulsando en la opción 3 de instalación:particionar el disco duro, podemos crear estas particiones. De todas formas, puede que te sientas más cómodo usando antes de empezar la instalación algún programa en windows como partition magic. el caso es crear estas 2 particiones antes de instalar el sistema.

Cuando ya tengamos las particiones damos a la opción 1, configurar la instalación. Donde rellenaremos una serie de datos necesarios para la instalación.

Elija tipo de sistema: elegimos begginer

elija partición en que instalar knoppix: elegimos la partición que hemos creado anteriormente, con cuidado de no equivocarnos, porque si elegimos otra se borrará todo lo que haya en ella.

Introduzca su nombre:pues eso :)

Después nos pedira un nombre de usuario y clave, que serán los que tengamos que escribir para acceder a linux una vez instalado.

Tras esto nos pide una clave de administrador. esta será la clave más importante, la que usemos para hacer cambios profundos en el sistema. instalar software...

Poner nombre a la máquina: el que quieras.

Después de rellenar todo esto ya lo tenemos todo. Queda instalar en el disco duro. Así que seleccionamos la opción 2-comenzar instalación.

Quiere proceder con estos parámetros?: comprobamos que todos los datos son correctos y damos al botón posterior

Entonces el sistema se empieza a copiar al disco duro. Esto lleva un tiempo, 20 minutos o así.

Ahora toca instalar el gestor de arranque en el MBR, si tenemos otros sistemas operativos como por ejemplo windows elegimos sí.

Al terminar nos pide un disquette para hacer un disco de arranque.

Y ya está. quitamos el cd arrancamos el cd, elegimos linux en el gestor de arranque y a configurar a nuestro gusto nuestro nuevo linux... muy fácil.
