RSS: Todas las noticias en tu ordenador

Qué es y cómo usarlo

Lunes 11 de Abril de 2005, por Álvaro García

RSS es un estándar que permite conocer las novedades de tus webs favoritas o las últimas noticias que han ocurrido en el mundo nada más ser publicadas. En vez de tener que ir a cada web para ver si hay novedades, puedes usar RSS (Really Simple Syndication) para obtenerlas cada vez que hay algo nuevo.

¿Qué son los archivos RSS?

Los archivos RSS son archivos XML utilizados para notificar las últimas novedades de una web. Estos archivos pueden ser leídos por programas especiales (lectores de noticias o titulares). En la wikipedia hay más definiciones sobre RSS (en inglés hay más información).

¿Cómo usarlo?

Cuando veas en una web Sindicar este sitio (Syndicate this site) y/o un dibujillo que ponga XML o RSS,puedes apuntarte la dirección de ese enlace y ver las noticias sin visitar la web, simplemente usando un programa lector de rss, también llamado agregador. A este enlace se le llama feed.

Existen muchos lectores de feeds RSS, yo uso feedreader. Es software libre, liberado bajo licencia GPL. Otros programas muy usados son: rssreader y sharpreader. También hay lectores on-line, páginas webs desde las que leer todos vuestros feeds, como por ejemplo www.bloglines.com

Para conocer las especificaciones técnicas de RSS puedes visitar alguna de estas webs: www.uatsap.com, www.mnot.net o www.w3.org

Feeds RSS

Los feeds son enlaces a los archivos rss, rpf o xml que le tienes que pasar a tu lector RSS para poder ver las novedades de la web en cuestión. Para poder empezar a leer noticias os voy a dar algunas direcciones con feeds RSS que me gustan:
